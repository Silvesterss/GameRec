# GameRec 🎮

> iOS 游戏推荐 App —— 推荐游戏本身，不是内容帖子

---

## ✨核心特性

- 🎯 **多平台账号绑定**：PSN / Steam / Switch三个平台并行绑定
- 📊 **多维数据分析**：游戏时长、进度、成就 综合利用
- 💰 **价格感知**：结合当前折扣推荐
- 🏷️ **两层推荐**：先推荐类别（RPG/Roguelike/Galgame...）→ 再推荐类内未玩游戏
- 🔄 **换一批**：类别换一批 + 游戏换一批
- 🔍 **可解释**：每个推荐都告诉你"为什么推荐它"

---

## 🎯 推荐算法（简述）

**第一层：类别推荐**
```
score_cat = α·偏好强度 + β·未覆盖度 + γ·价格友好度
```

**第二层：游戏推荐**
```
score_game = δ·内容相似度 + ε·价格友好度 + ζ·多样性惩罚
```

权重默认：`α=0.5, β=0.3, γ=0.2, δ=0.6, ε=0.2, ζ=0.2`
所有权重可在 `RecommendationWeights.swift` 中调整。
